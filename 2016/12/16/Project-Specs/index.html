Project Specs / Estimates

One thing I learned while working on the Advanced JavaScript project goal at the guild is that I should pay closer attention to project specs, and that I need to be more careful when estimating time. Initially my team mate had the idea to review each topic within the project spec, with the goal of 3 topics per day. This proved to be a very challenging goal to meet, and we were not able to cover all the topics we had in our project spec.

If I was going to work on this project goal again, I would remove many of the specs and put more focus on just a few topics (such as spending the full week just learning Node).
