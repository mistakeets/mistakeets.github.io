40 Weeks 40 Projects

Today was the first full day of working on a project. My time was spent reviewing html tags, CSS, and I learned the basics of using Bootstrap. In my opinion bootstrap feels kind of “messy” when you nest tags inside tags (inside tags!) as shown in this code example which creates labelled buttons:

<div class="container-fluid">
  <h3 class="text-primary text-center">jQuery Playground</h3>
  <div class="row">
    <div class="col-xs-6">
      <h4>#left-well</h4>
      <div class="well" id="left-well">
        <button class="btn btn-default target" id="target1">#target1</button>
        <button class="btn btn-default target" id="target2">#target2</button>
        <button class="btn btn-default target" id="target3">#target3</button>
      </div>
    </div>
    <div class="col-xs-6">
      <h4>#right-well</h4>
      <div class="well" id="right-well">
        <button class="btn btn-default target" id="target4">#target4</button>
        <button class="btn btn-default target" id="target5">#target5</button>
        <button class="btn btn-default target" id="target6">#target6</button>
      </div>
    </div>
  </div>
</div>

It took me a little while to get used to working with nested tags again, however working with Bootstrap was fun as I also learned you can use Font Awesome to apply a style or font to a web page using Bootstrap.

All in all I felt that working through some html, CSS, and Bootstrap tutorials was a useful refresher, however I also felt like I was not pushing myself hard enough.

My First Day

What a day! After orientation and meetings I chose a mentor and voted on my first project assignment here at the Learners Guild. Suddenly this whole new channel showed up in the chat application and I asked my mentor “Hey what the heck is this?!?!” Oh! I see… I have been assigned a team mate. This is an awesome process!

My team mate and I will work on the goal of HTML/CSS & JavaScript Basics this week. The majority of our time will be spent reviewing tutorials to ensure we have a solid foundation to build from going forward. This also feels like a good project work on during my first week as I need a refresher on basic JavaScript syntax. Before starting at the Guild I was taking an online course to learn the Python programming language, so now it is time to switch gears to become more comfortable with JavaScript again.

Let the games begin!
